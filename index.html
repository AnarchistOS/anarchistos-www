Features

New to AnarchistOS

  • AnarchistOS Linux is based on Crunchbang Plus Plus which is based on no longer maintained Crunchbang which was based on stable at the time Debian Jessie. ;-)
  • SLiM has been replaced by the light, and more standards-compliant LXDM.
  • Updated GTK+ 3.22 theme, and rebased CBPP icon theme on the latest Faenza set.
  • All new .iso's have a live boot option!
  • There will be netinstall available as well!
